	arr = [None] * 100

	random.shuffle(arr)

	x = 4950 #1+2+.....+99 = 4950

Bir tane dublicate sayı ekledim. Arrayi ekrana bastım. 'x' adında bir değişkene 4950 (1-99 arasındaki sayıların toplamı) değerini atadım.
